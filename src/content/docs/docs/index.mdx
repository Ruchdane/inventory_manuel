## Scénarios d'Utilisation

Découvrez comment notre application d'inventaire s'adapte à diverses situations pour répondre aux besoins des entreprises, petites ou grandes.

<CardGrid stagger>
  <Card title="Boutique Unique + Vente Directe" icon="store">
    Parfait pour les petites boutiques qui gèrent leur inventaire et vendent directement aux clients.
  </Card>
  <Card title="Magasin Unique + Plusieurs Boutiques + Ventes" icon="shopping-bag">
    Idéal pour les entreprises avec un point central de stockage (magasin) et plusieurs points de vente.
  </Card>
  <Card title="Plusieurs Magasins + Plusieurs Boutiques + Ventes" icon="storefront">
    Pour les grandes entreprises avec une structure complexe, notre application offre une gestion centralisée des stocks et une visibilité complète sur toutes les boutiques.
  </Card>
</CardGrid>
